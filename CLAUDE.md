# MATIWLOG - Proyecto de Marca Personal

## Contexto
Este es el sitio web y hub digital de **MATIWLOG** (Matias Aguilera), un creador de contenido de IA, piloto de drones, y CEO de Kreathur Agency, basado en Pichilemu, Chile.

## Stack Actual
- **Sitio web**: HTML/CSS/JS single-page (index.html)
- **Futura migracion**: Next.js 15 + Tailwind CSS + TypeScript
- **Hosting target**: Vercel o Cloudflare Pages
- **CMS**: Pendiente (considerar Sanity, Contentful, o MDX)

## Estructura del Proyecto
```
MATIWLOG/
├── index.html          # Landing page principal
├── CLAUDE.md           # Este archivo
└── agents/             # Sistema de agentes de mejora continua
    ├── seo-agent.md    # Instrucciones para agente SEO
    ├── content-agent.md # Instrucciones para agente de contenido
    └── design-agent.md  # Instrucciones para agente de diseno
```

## Principios de Diseno
- Dark mode con acentos cyan (#06d6a0) y amber (#f4a261)
- Tipografia: Syne (headings), DM Sans (body), JetBrains Mono (code/labels)
- Glassmorphism sutil, no exagerado
- Mobile-first, responsive
- Todo en espanol

## Modelo de Negocio
1. **Productos digitales**: Guias de IA, templates de n8n
2. **Consultoria**: 3 tiers ($15/$59/$199 USD)
3. **Contenido gratuito**: YouTube, Instagram (funnel de adquisicion)
4. **Pipeline a Kreathur Agency**: Para proyectos grandes
5. **Afiliados**: Herramientas de IA que Mati usa
