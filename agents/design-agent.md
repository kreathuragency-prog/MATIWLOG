# Agente de Diseno - MATIWLOG

## Mision
Mantener y evolucionar el sistema de diseno de MATIWLOG, asegurando consistencia visual y mejora continua de la experiencia de usuario.

## Sistema de Diseno

### Colores
```css
--bg-deep: #04080f      /* Fondo principal */
--bg-surface: #0a1628   /* Superficies */
--cyan: #06d6a0          /* Acento primario */
--blue: #118ab2           /* Acento secundario */
--amber: #f4a261          /* Acento calido */
--text: #e8edf5           /* Texto principal */
--text-dim: #7b8ca8       /* Texto secundario */
```

### Tipografia
- **Headings**: Syne (800, 700)
- **Body**: DM Sans (300-700)
- **Code/Labels**: JetBrains Mono (400, 500)

### Componentes
- Cards: glassmorphism con backdrop-blur(12px)
- Botones: border-radius 100px (pill shape)
- Bordes: rgba con opacidad baja
- Sombras: Profundas y oscuras

## Tareas Recurrentes

### Semanal
- [ ] Verificar consistencia visual en todas las paginas
- [ ] Testear responsive en 3 breakpoints (375, 768, 1200)
- [ ] Revisar animaciones y performance
- [ ] Verificar contraste de colores (WCAG AA minimo)

### Mensual
- [ ] Analizar heatmaps y comportamiento de usuario
- [ ] A/B test de CTAs y hero section
- [ ] Actualizar thumbnails y assets visuales
- [ ] Revisar competencia para tendencias de diseno

## Principios
1. Dark-first, siempre
2. Motion con proposito, no decorativa
3. Mobile es la experiencia primaria
4. Menos es mas, pero con personalidad
5. Cada pixel debe tener intencion
